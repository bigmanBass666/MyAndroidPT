# 移动应用开发（Android）实训

> 课程来源：人工智能学院 · 移动应用开发
> 课题：Android 注册登录模块
> 文档定位：6 个连续实训任务 + 1 份综合实训报告模板

本目录包含一份完整的 Android 注册登录模块实训材料。整体项目以「MyAndroidPT」项目为载体，从纯 UI 页面起步，逐步接入 MySQL 后端，再叠加 SharedPreferences 本地持久化能力，最终形成一个完整的"登录 / 注册 / 记住密码 / 自动登录"应用。

## 通用项目配置（贯穿多个实训）

下列配置在实训 1 之后会反复出现，先在这里汇总方便查阅：

- **项目名**：`MyAndroidPT`
- **包名模板**：`com.ljx.pt`（如 `com.ljx.pt1`）
- **MySQL 数据库名**：`androidPT`
- **数据表**：`Userinfo`（或 `userInfo`，实训中两种写法都出现过），`id` 为主键并自动递增
- **MySQL JDBC 驱动**：`mysql:mysql-connector-java:5.1.49`
- **MySQL 用户**：`root` / `123456`
- **关键权限**：`android.permission.INTERNET`

> 注意：原文中包名出现了 `com.ljx`（实训 1）和 `com.ljx.pt1`（实训 2 原文为 `com.lrq.pt1`，此处已统一为 `ljx`）两种写法。学生在做的时候应当把包名统一为 `com.ljx.pt`。
