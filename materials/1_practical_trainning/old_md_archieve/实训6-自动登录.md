## 项目描述

项目要求 Android Studio **自动登录**和**记住密码**的实现。

## 项目目标

- 熟练使用 `SharedPreferences` 存储用户账号和密码，以及是否记住密码和自动登录。
- 熟练使用 `CheckBox` 的 `OnCheckedChangeListener` 监听。

## 知识准备：SharedPreferences

> ⚠️ 原文误写为 `sharedPreferenced` / `SharePreferenced`，本文档统一为正确写法。

`SharedPreferences` 是 Android 中存储数据的一种方式，**采用键值对**存储数据。

### 使用过程

1. 获取 `SharedPreferences` 对象。
2. 调用 `edit()` 获取 `SharedPreferences.Editor` 对象。
3. 调用 `putBoolean()` / `putString()` 等向 `Editor` 对象添加数据。
4. 调用 `apply()` 提交数据。

## 实践步骤

### 步骤 1：实现两个 UI 联动逻辑

> 联动方向如下，用 `OnCheckedChangeListener` 监听两个 `CheckBox`：

| 操作 | 联动结果 |
|------|---------|
| 勾选"自动登录" | 自动勾选"记住密码" |
| 取消"记住密码" | 自动取消"自动登录" |
| 取消"自动登录" | 单独行为（"记住密码"状态由用户决定） |

> 📌 注意："勾选记住密码"**不**应自动勾选自动登录，这是用户主动选择。

### 步骤 2：在"记住密码"的基础上实现"自动登录"

逻辑要点：

- 登录成功时，若 `cbAutoLogin.isChecked()` 为 true：
  - 写入"自动登录"标志位到 SharedPreferences（`putBoolean("isAutoLogin", true)`）。
- 应用启动初始化登录页时：
  - 若 `isAutoLogin == true` → **跳过登录按钮点击**，直接用保存的账号密码进入欢迎页。
  - 若 `isAutoLogin == false` → 走正常流程。

```java
// 初始化时判断
SharedPreferences sp = getSharedPreferences("user_info", MODE_PRIVATE);
if (sp.getBoolean("isAutoLogin", false)) {
    // 直接进入欢迎页
    startActivity(new Intent(LoginActivity.this, WelcomeActivity.class));
    finish();
}
```

### 步骤 3：登出时取消"自动登录"

- 在欢迎页提供"退出登录"按钮。
- 点击后清空 SharedPreferences 中的"自动登录"标志位（账号密码可以保留，方便下次手动登录）。

```java
sp.edit().putBoolean("isAutoLogin", false).apply();
```

### 步骤 4：注册成功后自动回填登录页

#### 子步骤 (1) 注册成功时把用户名和密码打包回传

在 `RegisterActivity` 中：

```java
Intent data = new Intent();
data.putExtra("username", username);
data.putExtra("password", password);
setResult(RESULT_OK, data);
finish();
```

#### 子步骤 (2) 把原来的 `startActivity(intent)` 注释掉，改用数据回传

在 `LoginActivity` 中，把原跳转改成：

```java
// 注释掉
// startActivity(intent);

// 改为
startActivityForResult(intent, REQUEST_CODE_REGISTER);
```

#### 子步骤 (3) 数据回传后设置登录页面

在 `LoginActivity` 中重写 `onActivityResult`：

```java
@Override
protected void onActivityResult(int requestCode, int resultCode, @Nullable Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    if (requestCode == REQUEST_CODE_REGISTER && resultCode == RESULT_OK && data != null) {
        etUsername.setText(data.getStringExtra("username"));
        etPassword.setText(data.getStringExtra("password"));
    }
}
```

## 本实训小结

| 联动 / 能力 | 实现位置 |
|-------------|---------|
| 勾选"自动登录"→ 自动勾选"记住密码" | `LoginActivity` 中 `cbAutoLogin.setOnCheckedChangeListener` |

| 注册成功回填 | `RegisterActivity.setResult` + `LoginActivity.onActivityResult` |
| 数据持久化 | `SharedPreferences user_info` |
