## 项目描述

项目要求 Android Studio **记住密码**的实现。

## 项目目标

- 熟练使用 `SharedPreferences` 存储用户账号和密码，以及是否记住密码。
- 熟练使用 `CheckBox` 的 `OnCheckedChangeListener` 监听。

## 知识准备：SharedPreferences

> ⚠️ 原文将 `SharedPreferences` 误写为 `sharedPreferenced` / `SharePreferenced`，本文档统一为正确写法。

`SharedPreferences` 是 Android 中存储数据的一种方式，**采用键值对**存储数据。

### 使用过程

1. 获取 `SharedPreferences` 对象。
2. 调用 `edit()` 获取 `SharedPreferences.Editor` 对象。
3. 调用 `putBoolean()`、`putString()` 等向 `Editor` 对象添加数据。
4. 调用 `apply()` 提交数据。

```java
// 示例
SharedPreferences sp = getSharedPreferences("user_info", MODE_PRIVATE);
SharedPreferences.Editor editor = sp.edit();
editor.putString("username", etUsername.getText().toString());
editor.putString("password", etPassword.getText().toString());
editor.putBoolean("isRemember", true);
editor.apply();
```

## 实践步骤

### 步骤 1：登录成功后判断是否记住密码

在登录成功的回调里：

```java
if (cbRemember.isChecked()) {
    SharedPreferences sp = getSharedPreferences("user_info", MODE_PRIVATE);
    sp.edit()
      .putString("username", username)
      .putString("password", password)
      .putBoolean("isRemember", true)
      .apply();
}
```

### 步骤 2：下次进入应用时自动回填

> ⚠️ 原文把这条步骤编号为 "2、用于勾选了…"，是"步骤 2"的笔误，本文档修正为步骤 2。

当用户曾勾选过"记住密码"后，应用启动初始化登录页时，应**从 SharedPreferences 里把用户名和密码读回来**并填入输入框，同时把"记住密码"复选框设为已勾选。

```java
SharedPreferences sp = getSharedPreferences("user_info", MODE_PRIVATE);
if (sp.getBoolean("isRemember", false)) {
    etUsername.setText(sp.getString("username", ""));
    etPassword.setText(sp.getString("password", ""));
    cbRemember.setChecked(true);
}
```

## 本实训小结

| 行为 | 触发时机 | SharedPreferences 操作 |
|------|----------|------------------------|
| 写入账号密码 | 登录成功且勾选"记住密码" | `putString` × 2 + `putBoolean(isRemember, true)` + `apply()` |
| 读出账号密码 | 应用打开 / 登录页 `onCreate` | `getBoolean(isRemember)` → 决定是否回填 |
| 清空（登出时） | 用户主动登出 | `clear().apply()`，详见 [实训 6](./实训6-自动登录.md) |
