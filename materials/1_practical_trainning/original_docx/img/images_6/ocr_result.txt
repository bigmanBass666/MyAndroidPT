登录成功后 SharedPreferences 完整写入代码（含 auto-login 嵌套逻辑）

关键代码：
```java
public void onLoginSuccess(String name, String psw){
    // 勾选记住密码
    if(cb_pass_remember.isChecked()){
        SharedPreferences spf = getSharedPreferences("spfRecord", MODE_PRIVATE);
        SharedPreferences.Editor editor = spf.edit();
        editor.putString("userName", name);
        editor.putString("password", psw);
        editor.putBoolean("isRemember", true);

        // 勾选自动登录（嵌套在记住密码块内）
        if(cb_auto_login.isChecked()){
            editor.putBoolean("isAutoLogin", true);
        }else{
            editor.putBoolean("isAutoLogin", false);
        }
        editor.apply();
    }else{  // 没有勾选记住密码
        SharedPreferences spf = getSharedPreferences("spfRecord", MODE_PRIVATE);
        SharedPreferences.Editor editor = spf.edit();
        editor.putBoolean("isRemember", false);
        editor.apply();
    }

    // 跳到欢迎页面
    Intent intent = new Intent(MainActivity.this, WelcomeActivity.class);
    intent.putExtra("userName", name);
    startActivity(intent);
    MainActivity.this.finish();
}
```

命名规范确认：
- 新增 Key："isAutoLogin"（boolean）
- auto-login 判断嵌套在 isRemember=true 块内，非独立层级
- 方法名：onLoginSuccess() 或直接写在 onClick 里

Java 代码截图 — onActivityResult 方法（数据回传后设置登录页面）

关键代码：
```java
@Override
protected void onActivityResult(int requestCode, int resultCode, @Nullable Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    if(requestCode==1&& resultCode==1&& data!=null){
        Bundle bundle = data.getExtras();
        String userName = bundle.getString("userName","");
        String password = bundle.getString("password", "");
        etAccount.setText(userName);
        etPassword.setText(password);
    }
}
```

关键信息：
- 顶部注释：//数据回传后，设置登录页面
- 请求码判断：requestCode==1 && resultCode==1（红框标注）
- data!=null 非空校验
- Bundle 提取 userName 和 password
- etAccount / etPassword 为 EditText 控件（设置登录页面的文本）
- 方法名：onActivityResult
- void 返回类型（红框标注）
- 包名：com.example.mysqltest（红框标注顶部标签）
