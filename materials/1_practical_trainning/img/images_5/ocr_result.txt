=== img_02.png ===
登录成功后 SharedPreferences 写入 + 跳转 WelcomeActivity 的代码

关键代码：
```java
// 勾选记住密码 → 写入 spfRecord
if(cb_pass_remember.isChecked()){
    SharedPreferences spf = getSharedPreferences("spfRecord", MODE_PRIVATE);
    SharedPreferences.Editor editor = spf.edit();
    editor.putString("userName", name);
    editor.putString("password", psw);
    editor.putBoolean("isRemember", true);
    editor.apply();
}else{  // 没有勾选记住密码
    SharedPreferences spf = getSharedPreferences("spfRecord", MODE_PRIVATE);
    SharedPreferences.Editor editor = spf.edit();
    editor.putBoolean("isRemember", false);
    editor.apply();
}

// 跳转到欢迎页面
Intent intent = new Intent(MainActivity.this, WelcomeActivity.class);
intent.putExtra("userName", name);
startActivity(intent);
MainActivity.this.finish();  // 跳到欢迎页面后，当前的登录页面就销毁吧
```

命名规范（必须对齐）：
- spf 文件名："spfRecord"
- userName key："userName"
- password key："password"
- isRemember key："isRemember"
- CheckBox 变量：cb_pass_remember

=== img_04.png ===
initData() 方法截图

关键代码：
```java
private void initData() {
    SharedPreferences spf = getSharedPreferences("spfRecord", MODE_PRIVATE);
    Boolean isRemember = spf.getBoolean("isRemember", false);
    String userName = spf.getString("userName", "");
    String password = spf.getString("password", "");

    if(isRemember) {
        etAccount.setText(userName);
        etPassword.setText(password);
        cb_pass_remember.setChecked(true);
    }
}
```

命名规范（关键差异）：
- spf 文件名："spfRecord"
- isRemember key："isRemember"
- userName key："userName"
- password key："password"
- layout id：cb_pass_remember（非 cb_remember）
- 方法名：initData()（非 initDate()）
- 变量名 isRemember 首字母小写 Boolean
